# 变量

- 字符串
- 数字
- 布尔
- 数组
- 对象
- Undefined 和 null

```java
// 未定义类型
var x ;
// 数字类型
var y = 5;
// 字符串类型 双引号或者单引号均可
var k = 'john';
var a = 123e5;
// 布尔类型
var b = true;
// 数组类型
var cars = [];
cars[0] = "Saab";
cars[2] = "Volvo";
// 对象类型
var person = {name:"veng",age:22};
var name = person.name;
var name2 = person["name"];
```
